# Icon Interface

**Properties**

| Name   | Type                      | Description                                                         |
| ------ | ------------------------- | ------------------------------------------------------------------- |
| url    | string                    | The URL of the image.                                               |
| size   | [Size](#size-interface)   | The size of the image.                                              |
| anchor | [Point](#point-interface) | The point of the image that will correspond to the marker position. |
